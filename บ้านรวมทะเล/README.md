# AI Commerce Chatbot Showcase

A React + Tailwind mock-up website that demonstrates an AI commerce chatbot workflow with sanitized demo data.

## Features

- Interactive mock chat simulation
- Google Sheets style data sync demo
- Catalog carousel showcase
- Multi-platform automation overview for Facebook, Instagram, Telegram, and Line

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
