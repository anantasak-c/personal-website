# Turning Knowledge into Action: บทเรียนจาก 3 ปีในวงการ Tech

ผมเริ่มต้นในฐานะนักศึกษารัฐศาสตร์ที่ไม่รู้จัก Python เลยสักบรรทัด แต่วันนี้ผมทำงานด้าน Data Analysis, Blockchain และ Business Development ให้กับบริษัทที่ทำงานกับทีมข้ามประเทศ

คำถามที่หลายคนถามผมคือ **"เริ่มยังไง?"**

คำตอบสั้นๆ คือ: **ลงมือทำทันที ไม่รอให้พร้อม**

## บทเรียนที่ 1: เรียนเพื่อใช้ ไม่ใช่เรียนเพื่อรู้

ตอนเริ่มเรียน Python ผมไม่ได้นั่งอ่านหนังสือทีละบท แต่หยิบปัญหาจริงมาแก้ทันที เช่น ทำ script สำหรับ scrape ข้อมูลที่ต้องใช้ในงาน ทำให้เรียนรู้เร็วกว่าเดิมมาก

> ความรู้ที่ไม่ได้ใช้ จะเลือนหายไปภายใน 2 สัปดาห์

## บทเรียนที่ 2: Community คือ Shortcut ที่ดีที่สุด

การก่อตั้ง [KU Blockchain Society](https://github.com) ทำให้ผมเจอคนที่รู้มากกว่าผมในทุกด้าน เรียนรู้ผ่านการพูดคุย การจัดงาน และการแก้ปัญหาร่วมกัน ซึ่งเร็วกว่าการเรียน course คนเดียวหลายเท่า

## บทเรียนที่ 3: ความล้มเหลวคือ Data

ทุกโปรเจกต์ที่ไม่สำเร็จ คือข้อมูลที่บอกว่าอะไรควรทำต่างออกไป ผมไม่เคยมองว่าการล้มเหลวคือจุดสิ้นสุด แต่มองว่าเป็น iteration หนึ่งในกระบวนการ

---

## สิ่งที่ทำให้ผมอยู่รอดในวงการ Tech ทั้งที่ไม่ได้จบ CS

1. **เรียนรู้อย่างต่อเนื่อง** — ใช้เวลา 30 นาทีต่อวันกับสิ่งใหม่
2. **สร้าง portfolio จริง** — ไม่ใช่แค่ certificate
3. **รับงานที่ยากเกินตัวเล็กน้อย** — แล้วค่อยเรียนรู้ระหว่างทาง
4. **แชร์สิ่งที่รู้** — เพราะการสอนคือการเรียนรู้ที่ลึกที่สุด

ถ้าคุณกำลังอยู่จุดเดิมที่ผมเคยอยู่ — อยากเปลี่ยนสาย อยากเข้าวงการ Tech แต่ไม่รู้จะเริ่มจากไหน — คำแนะนำของผมคือ: **เริ่มจากปัญหาเล็กๆ ที่อยู่ตรงหน้าคุณวันนี้เลย**
